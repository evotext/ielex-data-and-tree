
data.csv
  The output of the make_ascertained_matrix.py script

ielex-130421-ag-cc.txt
  Raw data from Chang et al. 2015 corrections to the IELex database

longtable.csv
  ielex-130421-ag-cc.txt data transformed into XXX format (for use e.g. with XXX)

lookup.tsv
  table showing correspondences between language names (as used here and in
  Chang et al. 2015), glottocodes, and the language names used in IELex

make_ascertained_matrix.py
  a python script to transform raw data into beastling compatible format with a
  column marking the ascertainment correction (requires ielex-130421-ag-cc.txt
  and lookup.csv)
